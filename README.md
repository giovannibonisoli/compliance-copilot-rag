# RegGuard

RegGuard is a Retrieval-Augmented Generation (RAG) system for regulatory compliance assistance.

## Features

- PDF ingestion
- Text chunking with overlap
- Sentence Transformers embeddings
- Qdrant vector database
- Semantic search

## Tech Stack

- Python
- PyPDF
- Sentence Transformers
- Qdrant

## Current Status

- [x] PDF parsing
- [x] Chunk generation
- [x] Embedding generation
- [x] Vector indexing
